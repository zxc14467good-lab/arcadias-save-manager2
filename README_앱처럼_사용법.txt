# 아르카디아스 세이브 관리자 PWA v1

## 이게 뭐임?
폰에서 앱처럼 쓰기 위한 아르카디아스 세이브 관리자입니다.
브라우저로 열고 “홈 화면에 추가”하면 앱 아이콘처럼 실행할 수 있습니다.

## 안드로이드에서 앱처럼 쓰는 법

1. ZIP 압축을 푼다.
2. `index.html`을 연다.
3. Chrome 또는 삼성 인터넷에서 메뉴를 누른다.
4. “홈 화면에 추가” 또는 “앱 설치”를 누른다.
5. 홈 화면에 생긴 아이콘으로 실행한다.

## 주의
- 완전한 APK 앱은 아니고, 웹앱/PWA 방식입니다.
- 데이터는 휴대폰 브라우저 저장소(localStorage)에 저장됩니다.
- 브라우저 데이터 삭제 시 세이브가 사라질 수 있습니다.
- 그래서 가끔 “JSON 내보내기”로 백업하세요.
- Google Drive에 JSON 세이브를 백업하면 더 안전합니다.

## 파일 설명
- `index.html`: 앱 본체
- `manifest.json`: 앱 이름/아이콘/홈 화면 설정
- `service-worker.js`: 오프라인 캐시
- `icon.svg`: 앱 아이콘
